This is the source for https://rmoff.net, built using Hugo and hosted via GitHub pages.

== Running locally

[source,bash]
----
./serve.sh
----

Then visit http://localhost:1313. To use a different port: `./serve.sh 1314`.

See link:DEVELOPMENT.md[DEVELOPMENT.md] for full development documentation including CI/CD, link checking, and troubleshooting.
