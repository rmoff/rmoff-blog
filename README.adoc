This is the source for https://rmoff.net, built using Hugo and hosted via GitHub pages.

== Running locally

[source,bash]
----
./serve.sh
----

Then visit http://localhost:1313. To use a different port: `./serve.sh 1314`.

See link:DEVELOPMENT.md[DEVELOPMENT.md] for full development documentation including CI/CD, link checking, and troubleshooting.

== Most-clicked links (Interesting Links)

The link:https://rmoff.net/categories/interesting-links/[Interesting Links] pages surface their
most-clicked outbound links — a top-10 on the category landing page and a per-post top-5 in the
sidebar. Click data comes from PostHog (`il_link_clicked` events) and is read via a small
read-only Cloudflare Worker that keeps the PostHog API key server-side.

* Worker source, API reference, and deploy/local-dev steps: link:workers/popular-links/README.md[`workers/popular-links/`]
* The site reads the deployed Worker URL from `config.yaml` (`params.popularLinksEndpoint`). It is
  empty by default, and the widgets hide themselves gracefully until it is set, so the site is safe
  to build without the Worker.
