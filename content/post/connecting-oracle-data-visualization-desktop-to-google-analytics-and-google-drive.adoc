To use Data Visualisation Desktop (DVD) with data from Google Analytics or Google Drive, you need to set up the necessary credentials on Google so that DVD can connect to it. You can see a YouTube of this process on link:http://oracledataviz.blogspot.co.uk/2016/10/oracle-dv-connecting-to-google-analytics.html?ref=rittmanmead.com[this blog here].


Before starting, you need a piece of information from Oracle DVD that will be provided to Google during the setup. From DVD, create a new connection of type Google Analytics, and make a note of the the provided redirect URL:


image::/images/2016/11/ga01-1.png[]


Once you have this URL, you can go and set up the necessary configuration in Google. To do this, go to link:https://console.developers.google.com/?ref=rittmanmead.com[https://console.developers.google.com/] and sign in with the same Google credentials as have acces to Google Analytics.


Then go to link:https://console.developers.google.com/iam-admin/projects?ref=rittmanmead.com[https://console.developers.google.com/iam-admin/projects] and click on *Create Project*


image::/images/2016/11/ga12.png[]


Having created the project, we now need to make available the necessary APIs to it, after which we will create the credentials. Go to link:https://console.developers.google.com/apis/?ref=rittmanmead.com[https://console.developers.google.com/apis/] and click on *Analytics API*


image::/images/2016/11/ga13.png[]
